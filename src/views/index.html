North Loop Demo

- Invocation of 2 RapidAPI apis:

- Get Analysis: Get analysis of an underlying stock by symbol.

- Get News: Get the latest news of a stock by symbol. Please Note: This API is deprecated by RapidAPI but I have implemented as per the requirement.

- WebSocket: Backend logs are displayed on the right text box in real time via a websocket transport for the underlying logger.
